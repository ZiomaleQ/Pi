Declaration:

```

```

Comments:

```

```

If statement:

```
if (true) print("Hack");
 else print("Bruh");
```

Functions:

```
fun fib(n) {
    if (n <= 0) return 1; else return fib(n - 1) + fib(n - 2) 
}
```

Default argument:

```
fun fight(should = false) {
    if(should) print("We fight boiz");
    else print("We don't fight boiz");
}
```
